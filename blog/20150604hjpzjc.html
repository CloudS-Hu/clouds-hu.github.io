WAMP环境搭建教程

一、下载软件包:

  1. PHP
    php下载
    php-5.5.35-Win32-VC11-x64.zip  这里我们选择的是线程安全版
  2. MySQL
    MySQL下载
    Note: MySQL Installer is 32 bit, but will install both 32 bit and 64 bit binaries.
    mysql-installer-community-5.5.49.0.msi 安装版
    mysql-5.7.12-winx64.zip 免安装版
    有安装版和免安装版, 这里我们选择的是安装版.
  3. Apache
    apache下载
    http://www.apachelounge.com/download/
  4. VC11
    安装apache前必须先安装VC11（VSU_4\vcredist_x64.exe） VC11下载地址 选择英文版的

二、安装:

  1. 安装VC11，双击打开，点击若干个下一步直至安装成功就OK了；
  2. Apache 下载后是zip包，解压放在合适的位置就可以了，这里我们放在D:\PHPTools文件夹下的apache2.4目录下
  3. PHP下载后也是zip包，同样解压到合适位置，我们这里也将其放在D:\PHPTools文件夹下的php5.5目录下
  4. MySQL安装
    安装教程

三、配置(MySQL、PHP):

  1. 将D:\phptools\php5.5\php.ini-production 复制一份，并重命名为php.ini；
  2. 右键属性-->高级系 统设置 -->环境变量-->系统变量，找到Path，编辑，在其后加上 ;C:\Program Files\MySQL\MySQL Server 5.5\bin;D:\PHPTools\php5.5;D:\PHPTools\php5.5\ext

四、配置Apache:

  1. 用记事本打开F:\phptools\Apache2.4\Apache24\conf\httpd.conf 查找ServerRoot，修改ServerRoot "C:/Apache24" => ServerRoot "D:\PHPTools\apache2.4\apache24" （这里输入的是你解压apache安装包后放的位置）

  2. ServerName www.example.com:80 去掉

  3. DocumentRoot "c:/Apache24/htdocs" ， 修改为DocumentRoot "F:/phptools/Apache2.4/Apache24/htdocs"
  4. <Directory "c:/Apache24/htdocs"> ，修改为 <Directory " F:/phptools/Apache2.4/Apache24/htdocs ">
  5. 查找DirectoryIndex index.html ，添加index.php index.htm
  6. c/cgi-bin/ "c:/Apache24/cgi-bin/" ，修改为 ScriptAlias /cgi- bin/ "D:/PHPTools/apache2.4/apache24/cgi-bin/"
  7. <Directory "c:/Apache24/cgi-bin"> 修改为 <Directory "D:/PHPTools/apache2.4/apache24/cgi-bin">
  8. 在D:\PHPTools\apache2.4\apache24\conf\httpd.conf最后一行添加 LoadModule php5_module "D:/PHPTools/php5.5/php5apache2_4.dll" 让apache支持php（请确认D:/PHPTools/php5.5/有php5apache2_4.dll）
  9. 添加AddType application/x-httpd-php .php .html .htm
  10. PHPIniDir "D:/PHPTools/php5.5" （告诉apache php.ini的位置）
    httpd.conf 配置完成，保存httpd.conf

五、将Apache安装到系统服务中:

以管理员身份运行cmd.exe，cd到apache所在的磁盘D:， 输入D:\PHPTools\apache2.4\apache24\bin\httpd -k install回车安装

六、开启常用的PHP扩展:

记事本打开D:\PHPTools\php5.5\php.ini开启如下扩展 1、;extension_dir = "ext" 2、;extension=php_mbstring.dll（php多字节字符串扩展） 3、;extension=php_mysql.dll 4、;extension=php_mysqli.dll

七、启动apache和mysql:

双击D:\PHPTools\apache2.4\apache24\bin\ApacheMonitor.exe启动

八、搭建过程遇到的问题:

  1. 路由问题(一般存在下面三个问题, 如果还有出现其他情况 ,可以提出来一起研究):
    第一步检查httpd.conf中#LoadModule rewrite_module modules/mod_rewrite.so是否开启;
    第二步检查httpd.conf和httpd-vhosts.conf的AllowOverride None全部更改为All
    第三步检查.htaccess 文件设置RewriteEngine On是否为on
  2. openssl问题
    (1) 首先检查php.ini中；extension=php_openssl.dll是否存在， 如果存在的话去掉前面的注释符‘；’， 如果不存在这行，那么添加extension=php_openssl.dll。
    (2) 将php文件夹下的： php_openssl.dll， ssleay32.dll， libeay32.dll 3个文件拷贝到 WINDOWS\system32\ 文件夹下。
    (3) 如果没有 php_openssl.dll ，点击下载 php_openssl.dll 下载下来有很多个php_openssl，找到你相应php版本下面的php_openssl。 查看php版本(phpinfo()), 找到相应版本之后，按照上面讲的第二步来就可以了。
    (4) 重启apache至此，openssl功能就开启了。

九、其他配置:

  1. 配置apache根目录 在httpd.conf下找到 DocumentRoot "D:\PHPTools\apache2.4\apache24\htdocs" =>配置你的目录 <Directory "D:\PHPTools\apache2.4\apache24\htdocs"> =>配置你的目录 ... ...

  2. 配置虚拟主机, 在httpd.conf=》 #Virtual hosts #Include conf/extra/httpd-vhosts.conf（去掉#）

在httpd-vhosts.conf中进行配置

<VirtualHost *:80>
    DocumentRoot "E:/workspace/demo"
    ServerName demo.com
    DirectoryIndex index.php
    <Directory/>
    Options FollowSymLinks
    AllowOverride None
    Order allow,deny
    Allow from all
    </Directory>
</VirtualHost>
